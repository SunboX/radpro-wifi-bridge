RadPro WiFi Bridge Firmware Installer

Flash the latest RadPro WiFi Bridge firmware to an ESP32-S3 directly from your browser. Connect the board via USB, press the button below, and follow the prompts.

Supported hardware

The bundled firmware targets the ESP32-S3 DevKitC-1 (8 MB flash) that powers the RadPro WiFi Bridge. Make sure the board is in USB bootloader mode (hold BOOT while tapping RST) before starting the flash process if the device is not automatically detected.
